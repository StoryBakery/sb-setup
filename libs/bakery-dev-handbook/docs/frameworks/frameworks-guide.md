# Frameworks Overview

Story-Bakery 그룹의 프레임워크 모음은 그룹 어셋과 각 게임별 어셋을 구성할 때의 기준선이 되는 개발 규약과 공용 리소스를 제공합니다. 
이 문서는 프레임워크 전반의 방향성을 요약하고, 세부 가이드로 이동하기 전에 반드시 파악해야 할 핵심 원칙을 설명합니다.

- [scripting 가이드](./scripting/scripting-guide.md) 디렉터리는 코드 스타일, 네이밍, 객체지향 패턴 등 스크립팅 관련 세부 규칙을 다룹니다.
  스크립트 시에는 무조건 읽어보고 따라야할 규칙입니다.
- [instance-property-style](./instance-property-style.md) 인스턴스의 Property/Attribute 를 생성 및 사용할 때 무조건 읽어보고 따라야할 규칙입니다.
- [vide](./vide)는 StoryBakery 가 ui 를 만드는데 자주 사용하는 vide 에 대해 설명합니다
