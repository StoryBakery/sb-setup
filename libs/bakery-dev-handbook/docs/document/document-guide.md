

# Document Guide

문서를 어떻게 작성하는 지에 대한 가이드입니다.

## 문법

문장 마지막에 점은 무조건 찍습니다.

## 헤더

모든 문서에는 `# 개요`를 가장 맨 위에 적습니다.

## 텍스트-스타일링-금지 
볼드 처리를 할 때 헤더 제목 전체를 볼드 처리를 하지 않습니다.

```md
-- Bad:
# **볼드-헤더**

-- Good:
# 그냥-헤더
```

## 공백은-대시로

단어와 단어 사이의 공백은 대시(`-`)로 표기합니다.

```md
-- Bad:
# 문서 작성 가이드

-- Good:
# 문서-작성-가이드
```

이유는 마크다운 문서가 HTML 로 변환될 때 공백이 `%20` 으로 바뀌어 URL 이 길어지는 현상을 방지하기 위함입니다.

## 링크

문서 내 링크는 반드시 현재 문서를 기준으로 한 상대 경로(`../path/to/doc.md`)로 작성합니다. 저장소 루트 절대 경로나 외부 URL을 사용하는 방식은 허용하지 않습니다.
